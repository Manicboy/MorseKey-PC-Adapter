# MorseKey-PC-Adapter

#### Used tools:
* Arduino IDE 1.8.19 (https://www.arduino.cc/)
* Teensyduino 1.56 with Audio Lib (https://www.pjrc.com/)

#### Important:
USB type must set to "Serial + MIDI + Audio"
